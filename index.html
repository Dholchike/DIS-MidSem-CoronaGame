How To Play

Start the game by pressing SPACEBAR. Use the UP
arrow key to avoid the coronavirus getting hit in the vaccine.
The objective of the game is to gain the maximum score.

About The Game

This game has been made in remembrance of the horrors of
coronavirus that 2020 brought along with it. The coronavirus
wants to infect as many people as it can, meanwhile
vaccines as obstacles are placed, which the player has to avoid.
This game draws its inspiration from the Chrome-Dino/T-Rex Game.
